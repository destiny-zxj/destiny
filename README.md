## Usage
* Start Downie4 download through custom URL. Of course, first of all, you need to have a Downie4.
> open the url in browser
> destiny://cmd=downie4.download&url=https://www.xxx.com/video.m3u8&title=VideoTitle
```bash
cmd: constant value
url: video url
title: video title
```
